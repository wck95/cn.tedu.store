#学子商城项目

1.该项目使用springboot+mybatis搭建，是个单体结构的项目！

2.该项目功能有：
   |--- @已完成模块
   |  |
   |  |------|--->[  1：登录/注册   ]
   |  |------|--->[  3：收货地址    ]
   |  |------|--->[  4：商品展示    ]
   |  |------|--->[  5：收藏夹      ]
   |  |------|--->[  6：购物车      ]
   |  |------|--->[  7：订单生成    ]
   |  |
   |---@未完成模块
   |  |
   |  |------|--->[ 2.8：支付模块     ]
   |  |------|--->[ 2.9：vip模块      ]
   |  |------|--->[ 2.10：超市        ]
   |  |------|--->[ 2.11：优惠卷      ]
